BalancedFixMatch
========

BalancedFixMatch is the model to classify COVID-19 based on Chest X-ray images.

## Requirements

* Python 3.7
* PyTorch 1.7.0

## Datasets

* Chest X-ray dataset

  * [COVIDx-CXR dataset](https://github.com/lindawangg/COVID-Net/blob/master/docs/COVIDx.md)
